# Как запустить и отправить заявку (без ошибок)

## Важно: откуда открывать сайт

Сайт нужно открывать **только по адресу** в браузере:
- **http://127.0.0.1:5173**  
или  
- **http://localhost:5173**

Не открывай папку с диска (не двойной клик по index.html) и не папку dist — иначе будут ошибки и форма не заработает.

---

## Шаг 1. Остановить всё

1. В терминале, где запущен бэкенд: нажми **Control + C** (остановить).
2. В терминале, где запущен фронтенд (npm run dev): нажми **Control + C** (остановить).
3. В браузере закрой вкладку с приложением (или закрой все вкладки с 127.0.0.1 / localhost).

---

## Шаг 2. Запустить бэкенд

В терминале выполни (из корня проекта или из папки frontend):

```bash
cd backend
node server.js
```

Должно появиться: **Backend ready on http://localhost:3001**  
Этот терминал не закрывай.

---

## Шаг 3. Запустить фронтенд

Открой **второй** терминал и выполни:

```bash
cd frontend
npm run dev
```

Должно появиться что-то вроде: **Local: http://127.0.0.1:5173/**  
Этот терминал тоже не закрывай.

---

## Шаг 4. Открыть сайт в браузере

1. Открой браузер (Chrome, Safari, Firefox).
2. В адресной строке введи **ровно**: **http://127.0.0.1:5173**
3. Нажми Enter.

Должен открыться каталог товаров (серые карточки «Товар»).  
Если видишь старые цветные картинки с интернета — сделай жёсткое обновление: **Cmd+Shift+R** (Mac) или **Ctrl+Shift+R** (Windows).

---

## Шаг 5. Отправить заявку

1. Нажми «Узнать цену» на любом товаре (можно на нескольких).
2. Вверху справа нажми кнопку **«Выбрано: 1»** (или сколько выбрал).
3. В модалке нажми **«Запросить КП»**.
4. Заполни форму (имя, телефон, email и т.д.) и поставь галочку согласия.
5. Нажми **«Отправить»**.

Если всё настроено правильно, появится сообщение **«Заявка отправлена!»** и сообщение придёт в Telegram.

---

## Если снова ошибка

В алерте теперь будет подсказка. Пришли **полный текст** из этого алерта — по нему можно будет понять причину.

Ошибки вида **(ffffff, line 0)** в консоли — это старые запросы к картинкам. После перезапуска по шагам выше и открытия **http://127.0.0.1:5173** их быть не должно. Если они всё ещё есть — сделай **Cmd+Shift+R** (жёсткое обновление страницы).
