## Форматы

Подойдут **jpg**, **png** или **webp**. В данных каталога сейчас указан `.jpg` — если используете другой формат, нужно будет поменять расширение в `src/data/products.ts` для нужного товара (или оставить .jpg и просто переименовать файл).

## Если файла нет

Пока файла нет в папке, в карточке будет показываться плейсхолдер. Добавьте файл с нужным именем — после обновления страницы появится ваше фото.
