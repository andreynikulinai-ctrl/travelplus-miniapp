В терминале выполняй **по одной команде** (скопируй строку → вставь в терминал → Enter):

```
cd /Users/andreynikulin/Desktop/travelplus-miniapp
```

**Команда 2:**
```
git add -A
```

**Команда 3:**
```
git commit -m "Бот: webhook приветствие и кнопка Старт"
```

**Команда 4:**
```
git push origin main
```

Дождись, пока Vercel соберёт проект (1–2 минуты). Статус смотри на vercel.com → твой проект → Deployments.

## Шаг 2. Узнать домен и токен бота

- **Домен** — адрес твоего сайта на Vercel, например: `travelplus-miniapp.vercel.app` (посмотри в Vercel → проект → Domains или в последнем деплое).
- **Токен бота** — уже есть в настройках Vercel (переменная `TELEGRAM_BOT_TOKEN`). Его же дал @BotFather. Если не помнишь — зайди в Telegram → @BotFather → /mybots → выбери бота → API Token.

---

## Шаг 3. Один раз установить webhook

Открой в браузере ссылку (подставь вместо `<ТОКЕН>` и `<ДОМЕН>` свои значения):

```
https://api.telegram.org/bot<ТОКЕН>/setWebhook?url=https://<ДОМЕН>/api/telegram-webhook
```

**Пример** (если домен `travelplus-miniapp.vercel.app`, токен `123456:ABC-def`):

```
https://api.telegram.org/bot123456:ABC-def/setWebhook?url=https://travelplus-miniapp.vercel.app/api/telegram-webhook
```

Должен открыться ответ вроде: `{"ok":true,"result":true,"description":"Webhook was set"}`. Тогда всё ок.

---

## Шаг 4. Проверить бота

1. Открой своего бота в Telegram.
2. Нажми **Start** (кнопка внизу в чате) или напиши `/start`.
3. Должно прийти приветствие и под ним большая кнопка **Старт**.
4. Нажми **Старт** ещё раз — снова должно прийти приветствие (писать /start не нужно).
