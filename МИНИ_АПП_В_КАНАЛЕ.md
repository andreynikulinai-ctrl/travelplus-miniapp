## Шаг 1. Узнать URL приложения (Vercel)

1. Зайди на [vercel.com](https://vercel.com) и открой свой проект.
2. Открой последний деплой (или нажми **Visit** у продового домена).
3. Скопируй адрес из браузера — это твой URL приложения.  
   Пример: `https://travelplus-miniapp.vercel.app` (у тебя может быть другой).
4. Запомни или вставь сюда: **________________________________** (подставь свой URL).

Если проект ещё не задеплоен: сделай `git push` — Vercel соберёт и даст URL.

---

## Шаг 2. Привязать приложение к боту в BotFather

1. Открой Telegram и найди **@BotFather**.
2. Отправь команду: **`/mybots`**
3. В списке выбери бота **@travelplus_shop_bot** (нажми на него).
4. Нажми **«Bot Settings»** (Настройки бота).
5. Нажми **«Menu Button»** (Кнопка меню).
6. Нажми **«Configure menu button»** (Настроить кнопку меню).
7. BotFather попросит ввести **URL**.  
   Вставь свой URL приложения (из шага 1), например:  
   `https://travelplus-miniapp.vercel.app`  
   (обязательно **https**, без слэша в конце).
8. Когда попросит **текст кнопки**, введи, например: **`Каталог`** или **`Открыть каталог`**.
9. Готово. BotFather подтвердит настройку.

**Проверка шага 2:**  
- Зайди в чат с ботом **@travelplus_shop_bot**.  
- Внизу рядом с полем ввода должна быть кнопка меню (например «Каталог»).  
- Нажми её → должно появиться окно «Открывая это мини-приложение…» с кнопками **Отмена** и **Запустить**.  
- Нажми **Запустить** → откроется твой каталог внутри Telegram.

---

## Шаг 3. Добавить кнопку «Мини-апп» в канал @travelplusbiz

1. Открой канал **@travelplusbiz** (ты должен быть админом).
2. Создай **новый пост** или открой пост, который будешь редактировать.
3. В редакторе поста найди возможность **добавить кнопку** под постом (часто: иконка ссылки / «Add link» / «Включить кнопку»).
4. Задай:
   - **Текст кнопки:** например **`Открыть каталог`** или **`Каталог Трэвел+`**.
   - **Ссылка:** вставь ровно эту ссылку:  
     **`https://t.me/travelplus_shop_bot/app`**
5. Опубликуй (или сохрани) пост.

**Проверка шага 3:**  
- Открой канал **@travelplusbiz** (можно с другого аккаунта или в режиме просмотра).  
- Нажми на кнопку под постом (например «Открыть каталог»).  
- Должно открыться то же окно «Открывая это мини-приложение…» → **Запустить** → каталог внутри Telegram.

## Кратко

| Действие | Где | Что ввести |
|----------|-----|------------|
| URL приложения | Vercel → Visit | Твой домен, например `https://....vercel.app` |
| Menu Button → URL | BotFather → @travelplus_shop_bot → Menu Button | Тот же URL из Vercel |
| Menu Button → текст | BotFather | `Каталог` |
| Кнопка в канале | Пост в @travelplusbiz | Ссылка: `https://t.me/travelplus_shop_bot/app` |

Если на каком-то шаге что-то не совпадает (нет кнопки, другая ссылка) — напиши, на каком шаге и что видишь, подправим.
